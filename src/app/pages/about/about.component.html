ng2-dinos is a sample application built with Angular 2 with the following features:

  • Routing
  • Dynamic <title> metadata
  • External GET API
  • Custom off-canvas navigation
  • Filtering by predicate
  • Bootstrap
  • SCSS
  • Angular CLI (Webpack) build

Download the code for this app from the ng2-dinos GitHub repo. The API can be downloaded from the sample-nodeserver-dinos repo. The purpose of this project is to demonstrate an AngularJS 1.x app (without backported v2 features) "migration"/translation to Angular 2.
